# Road-accident-analysis
# 🚗 Smart Road Accident Analysis & Prediction System

## Overview

This project provides a comprehensive analysis of road accident-related data. Using Python and popular data analysis libraries, we explore various factors influencing accidents such as speed, weather conditions, and time. The project also uses machine learning techniques to identify patterns and predict accident risks.

---

## Dataset

The dataset contains structured information related to road accidents, including:

* Speed
* Speed_Limit
* Weather
* Time_Category
* Hour

---

## Features

### Data Cleaning & Preprocessing

* Removal of duplicate entries
* Handling missing values
* Data type conversion and formatting

### Exploratory Data Analysis

* Dataset overview (shape, columns, data types)
* Statistical summaries
* Data visualization

### Accident Insights

* Speed Distribution (Histogram)
* Weather vs Speed (Box Plot)
* Time vs Speed (Line Plot)
* Speed vs Speed Limit (Scatter Plot)
* Hourly Accident Trends (Bar Chart)

### Advanced Analytics

* Correlation analysis using heatmap
* Identification of high-risk conditions
* Pattern analysis across variables

## Usage

* Open the Jupyter Notebook (.ipynb file)
* Place dataset in the same directory
* Run all cells to:

  * Load and clean data
  * Perform analysis
  * Generate visualizations
  * Train machine learning model

---

## Key Findings

* Overspeeding is a major cause of accidents
* Bad weather increases accident risk
* Night and evening are high-risk periods
* Speed limit violations are common

---

## Visualizations

* Box plots for weather analysis
* Scatter plots for speed comparison
* Heatmap for correlation
